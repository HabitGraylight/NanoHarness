# NanoHarness

NanoHarness is a lightweight Python project scaffold for building and testing harness-style tooling.

## Features

- Minimal Python project layout
- Separate folders for configs, examples, and tests
- Ready for further development and version control

## Project Structure

```text
NanoHarness/
├── configs/
├── examples/
├── nanoharness/
├── tests/
├── main.py
└── requirements.txt
```

## Getting Started

### Requirements

- Python 3.10+

### Install Dependencies

```bash
pip install -r requirements.txt
```

## Run

```bash
python main.py
```

## Development

- Put core package code in `nanoharness/`
- Add runnable examples in `examples/`
- Add automated tests in `tests/`
- Store project configuration under `configs/`
